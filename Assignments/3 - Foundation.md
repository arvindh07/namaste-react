# Theory

## what is JSX and superpowers of jsx?

- jsx is a syntax where we can write js and html-like code in one place.
- it gives an enormous advantage of building app faster.

## Role of type attribute in script tag? What options can I use there?

- The type attribute in a script tag specifies the scripting language used within that element. Common values include:
     - "text/javascript" for JavaScript (default).
     - "text/babel" for JSX when using Babel for transpilation.
